1. FizzBuzz

Program ini menggunakan perulangan dan percabangan untuk menampilkan angka 1-100 dengan aturan tertentu. 



    
Code Python:

    
for i in range(1, 101):
    if i % 3 == 0 and i % 5 == 0:
        print("FizzBuzz")
    elif i % 3 == 0:
        print("Fizz")
    elif i % 5 == 0:
        print("Buzz")
    else:
        print(i)
    


    
Code JavaScript:

    
for (let i = 1; i <= 100; i++) {
    if (i % 3 === 0 && i % 5 === 0) {
        result += "FizzBuzz";
    } else if (i % 3 === 0) {
        result += "Fizz";
    } else if (i % 5 === 0) {
        result += "Buzz";
    } else {
        result += i;
    }
}
    







    
2. Konversi Hari


    

    Program ini mengubah jumlah hari menjadi tahun, bulan, dan hari menggunakan 
    operasi pembagian dan sisa bagi.

bulan = sisa_hari // 30
hari = sisa_hari % 30

print("Bulan :", bulan)
